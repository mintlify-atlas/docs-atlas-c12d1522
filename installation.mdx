# Installation

Simple Alarm Clock is available on both Google Play and F-Droid. Choose your preferred installation method below.

## Requirements

<Note>
  Simple Alarm Clock requires Android 5.0 (Lollipop, API level 21) or higher. The app targets Android 13 (API level 33).
</Note>

## Installation options

<Steps>
  <Step title="Choose your app store">
    Select Google Play for the latest version with automatic updates, or F-Droid for a fully open source installation.
  </Step>
  
  <Step title="Install the app">
    Follow the installation steps for your chosen platform below.
  </Step>
  
  <Step title="Grant permissions">
    After installation, you'll need to grant certain permissions for the app to function properly.
  </Step>
</Steps>

### Google Play

The recommended way to install Simple Alarm Clock is through Google Play:

1. Visit the [Simple Alarm Clock page on Google Play](https://play.google.com/store/apps/details?id=com.better.alarm)
2. Tap **Install**
3. Wait for the download and installation to complete
4. Open the app and grant the required permissions

The Google Play version includes automatic updates and is signed with the official release key.

### F-Droid

For users who prefer fully open source software, Simple Alarm Clock is available on F-Droid:

1. If you don't have F-Droid installed, download it from [f-droid.org](https://f-droid.org/)
2. Open F-Droid and search for "Simple Alarm Clock"
3. Alternatively, visit the [direct F-Droid link](https://f-droid.org/packages/com.better.alarm/)
4. Tap **Install**
5. Open the app and grant the required permissions

<Note>
  F-Droid builds may take a few days to update after a new release since they are built independently from source.
</Note>

## Required permissions

Simple Alarm Clock requires the following permissions to function properly:

| Permission | Purpose |
|------------|----------|
| **SCHEDULE_EXACT_ALARM** / **USE_EXACT_ALARM** | Schedule alarms to fire at precise times |
| **VIBRATE** | Enable vibration for alarms |
| **WAKE_LOCK** | Keep the device awake when an alarm fires |
| **DISABLE_KEYGUARD** | Show alarm over the lock screen |
| **RECEIVE_BOOT_COMPLETED** | Restore alarms after device reboot |
| **READ_MEDIA_AUDIO** | Access audio files for custom alarm sounds |
| **USE_FULL_SCREEN_INTENT** | Display full-screen alarm notifications |
| **POST_NOTIFICATIONS** | Show alarm notifications (Android 13+) |

<Warning>
  On Android 12 and later, you must explicitly grant the "Alarms & reminders" permission for the app to set exact alarms. The app will prompt you to grant this permission when needed.
</Warning>

## Application ID

The app uses the package name `com.better.alarm` for both Google Play and F-Droid versions.
